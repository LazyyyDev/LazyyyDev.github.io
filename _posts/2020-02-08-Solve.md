## 문제
```
---
layout: post
title: "첫번째 포스트"
tags: Blog
comments: false
---
```
위와 같이 comments를 비활성화를 해도 여전히 댓글이 생긴다...ㅜㅜ   

---

## 원인
가만보니 `post.html` 파일을 들여다 보면 `disqus.html` 을 불러오는 구문이 있는데
```
 {% include disqus.html %}
```
이렇게 되어있어서 그런 것 같다. 그냥 comments의 값과는 별개로 반드시 댓글창을 만들어내더라..   

---

## 해결
그래서 나는 아래와 같이 comments 상태값을 넘겨 이러한 문제를 해결했다.
```
 {% if page.comments %}
  
    {% include disqus.html %}
  
  {% endif %}
```
